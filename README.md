This project run on Python 3.9.2

# Step to run project
## Step 1: Install dependencies
Run this in cmd
```
pip install -r requirements.txt
```

## Step 2: Change model (Optional)
- Move your model trained to `models` folder
- In `utils.py` file, change name to your model 

## Step 3: Run project
Run this in cmd
```
py app.py
```
